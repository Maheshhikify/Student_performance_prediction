# Student_performance_prediction

##  Overview
The Student Performance Prediction System is a Machine Learning project that predicts whether a student will pass or fail based on academic and behavioral data.

---

##  Problem Statement
Students at risk are often identified too late, leading to poor academic performance and increased dropout rates.

---

## Solution
This project builds a predictive model to:
- Analyze student data  
- Predict Pass/Fail outcomes  
- Identify at-risk students early  

---

##  Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  
- Seaborn  

---

## Dataset
Synthetic dataset generated using NumPy.

### Features:
- Study Hours  
- Attendance (%)  
- Previous Marks  
- Assignment Scores  

### Target:
- 0 → Fail  
- 1 → Pass  

---

## ⚙️Project Workflow
Data → Preprocessing → Model → Prediction → Output

---

## 📈 Model Used
Random Forest Classifier
